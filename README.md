### HOW TO RUN

#### REQUIREMENT

```

 - expo > 3.2
 - node > 12.1
 - react-native-cli: > 2.0
 - yarn > 1.2

 IF YOUR RUN EMULATOR
    - openjdk version > 1.8.0

```

#### STEP 1
```
    git clone git@github.com:tienntps09110/react-native-proq-school.git

    yarn install
```

##### STEP 2
```
    CHOOSE HOW TO OPEN
    - yarn start
    - expo start
    - expo start --android
    - expo start --ios
    - expo start --web
    - expo web
    - expo eject
```
### MORE
```
- Set lan: set REACT_NATIVE_PACKAGER_HOSTNAME= ip_lan
```
